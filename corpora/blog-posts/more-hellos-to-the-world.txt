Here are some more "Hello World" examples in various languages that I've learned
(at least to a small extent):

// Javascript
document.write('Hello, World!');


// Java
public class HelloWorld
    public static void main(String[] args) {
        System.out.println('Hello, World!');
    }
}


// C
#include <stdio.h>
int main(void) {
    printf('Hello, World!
');
}


/* CSS */
body::before {
    content: 'Hello, World!';
}


// Ruby
puts 'Hello, World!'


# Bash
echo 'Hello, World!'


<?php
    // PHP
    echo 'Hello, World!';
?>


-- SQL
SELECT 'Hello, World!'


(These were not selected in any particular order.)
